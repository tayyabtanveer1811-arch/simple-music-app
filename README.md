# 🎵 Simple Music App

A beginner-friendly Flutter project where users can play different sounds by tapping on containers. This project was built as part of a university assignment.
## 🚀 Features
- Splash screen on app launch
- Tap on containers to play sounds
- Simple and interactive UI
## 🛠️ Tech Stack
- Flutter
- Dart
